## 🚀 كيفية التشغيل

### 1. تشغيل Backend
```bash
cd Backend/ArchitectPOS.API
dotnet restore
dotnet ef database update
dotnet run
```

### 2. تشغيل Frontend
```bash
cd Frontend
npm install
npm start
```
